# Hello!

I once built a bunch of wiz-bang ETW reactive extension things that used TDH APIs and did all sorts of craziness.  Additionally, I had a bunch of colleagues doing similar things with ETW and Evtx with RX memory streams.

This is a simple C# repo that swiftly reproduces some very neat ETW memory stream concepts off the back of those colleague's work.
